浮躁之年

Oct 05 2015

这个社会是浮躁的。这句话我是认同的。
微博的兴起就是浮躁的表现之一，人们不再有耐心去看超过一百四十个字以上文字的内容，当然，也不愿意去写超过一百四十个字的内容了。因为这是一个快节奏的时代，什么都讲究高效，快速，所有内容也呈现碎片话，具像化，有电影版本的，绝对不看电视剧版本，有电视机版本绝对不去看原著，人们懒得去自己构想一个书中的人物，甚至害怕没有影像声音的世界，只有铅字的书页会让他们产生不安，恐惧，还没翻看几页，已经焦躁不安。
上面说的人具有代表性，当然不是全部，我也是其中之一。

解决 误按 command ＋ q

Oct 03 2015

习惯使用快捷键切换，效率。
但最近经常使用 Chrome 浏览器切换标签页的时候，比如经常切换到的第一个标签页 command ＋ 1. 但是稍不注意，或者说小手够不着，哈哈，会没按到1，却按倒了 q , 但是大家都知道，command + q 是系统默认强制退出的快捷键，所以会把你所有打开的标签页全部关闭，比如你看东西看了一半，或者邮件写了一半，或者更衰一点，你用隐身模式开启了很多标签页，那你瞬间就想哭了。
