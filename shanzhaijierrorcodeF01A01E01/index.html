山寨手机出现 Error code F01A01E01 解决方法

手机出现的问题
为什么会接触到这个问题，邻居大姐手机出现这个问题，所以找上门来。一看错误代码 Error code F01A01E01，没见过。

尝试双清

所以准备先双清回复出厂设置，也不常弄，所以忘了怎么具体哪两个按键是进入 Rec ，不小心变成了重启手机。

发现是山寨手机

紧接着的开机画面 ———— 金来手机！靠山寨起家的金立，应该万万没想到自己也会被仿。瞬间就感觉到今天这个手机没那么好弄了，山寨机一般都没有刷机包，也不会有 Rec ，只能 Google 找找方法了。

寻找解决办法并尝试

搜索到的方法大部分都是针对正规品牌机，通过重刷系统，或者双清，或者通过刷机大师 刷机精灵之类的工具安装一个不能删除的应用，一般反映都是叫 卓易市场 的这个应用。但是上面已经说了，刷机和双清是不可能了，只能试试刷机大师和刷机精灵，但是当我下载好两个软件之后，由于手机被那个错误代码提示的弹窗挡住，都没法点击 USB调试 ，更别说要通过他们去安装一个应用到手机了。

总结遇到的情况

既然原因已经了解了，就是把 卓易市场 这个不能删除的应用重新装到手机上就可以解决，那么不能用电脑传输安装包的情况下，手机开机后也只有几秒的时间可以进行操作，否则就会被错误代码弹窗阻挡。那么就是只要把安装包弄到手机里，并在几秒钟内迅速安装就可以解决问题。

想到不同的解决思路

接下来，安装包可以通过 SD卡 的形式加入手机存储，为了更快的进行安装，最好放在更目录，并且在手机重启开机后的几秒内，迅速打开文件管理，找到 SD卡 下的 卓易市场 安装包，点击安装，在安装的过程中，那个错误代码弹窗应该会又跳出来，但是此时安装已经进行，等待安装完成，重启手机，问题应该就解决了。

最后总结

虽然大部分攻城狮不会用这种手机，但是你也有可能会遇到我这种情况，你的邻居或者你的亲戚之类的，如果遇到这种问题，希望这篇文章对你有用。
