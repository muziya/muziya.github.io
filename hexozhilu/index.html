Hexo

Sep 17 2015

为了学习前端，并且能够记录下学习过程和过程中遇到的问题及解法，所以一直想搭建一个博客。
兜兜转转用了Hexo，可能因为它更Geek一点，也不需要去搭建服务器，不用去想一个域名，也是免费的。
开始之前，看了大量的介绍，所以觉得应该半个小时应该可以搞定，但有时候，事情并不那么顺利，我就把我碰到的以及与官网不同的方法说一下。

本地安装一个适合Hexo的环境

  1. Hexo 官方文档说了，两个程序，node.js git
  2. 因为我是 Mac 用户，官网介绍的方法是通过 Xcode 命令行，可是当我打开 App Store的时候，发现它居然有 2.6G，就我的这个带宽环境想想还是算了。
  3. 所以我用了最简单的方法，就是到 node.js 官网 和 git 下载地址 下载安装包，直接下一步下一步安装了。
  4. 环境解决好，就该开始主题，安装Hexo。
